# UMC-CarrotMarketClone

UMC 3기 활동을 하며 배운 내용으로 당근마켓 클론 코딩을 진행한 프로젝트입니다.
<br><br>

## ✨ 결과물
[mingsound.shop](https://mingsound.shop/)
<br><br>

## ✔ 개발 스택
Front - HTML, CSS, JS, BOOTSTRAP <br>
Back - SpringBoot <br>
DB - MySQL <br>
<br>

## 📌 주요기능
- 회원가입
- 로그인/로그아웃
- 회원 조회/수정/삭제
- 게시물 작성/조회/수정/삭제
<br>
